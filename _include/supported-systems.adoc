|===
| 环境 | 协议 | 支持的版本 


| Amazon FSx for NetApp ONTAP | CIFS、NFS 和 SAN | 不适用 


| Azure NetApp Files | CIFS 和 NFS | 不适用 


.2+| 适用于 AWS 的Cloud Volumes ONTAP | CIFS 和 NFS | 9.11.1 及更高版本 


| SAN（iSCSI 和 NVMe） | 9.17.1 及更高版本 


.2+| 适用于 Google Cloud Platform 的Cloud Volumes ONTAP | CIFS 和 NFS | 9.11.1 及更高版本 


| SAN（iSCSI 和 NVMe） | 9.17.1 及更高版本 


.2+| 适用于 Microsoft Azure 的Cloud Volumes ONTAP | CIFS 和 NFS | 9.12.1 及更高版本 


| SAN（iSCSI 和 NVMe） | 9.17.1 及更高版本 


.2+| ONTAP （本地部署） | CIFS 和 NFS | 9.11.1 及更高版本 


| SAN（iSCSI 和 NVMe） | 9.17.1 及更高版本 


| Google Cloud NetApp Volumes | CIFS、NFS、SAN (iSCSI & NVMe) | 不适用 
|===

NOTE: Azure NetApp Files 或 Google Cloud NetApp Volumes 不支持用户活动检测。有关支持用户活动检测的详细信息，请参见 link:reference-user-activity-requirements.html["用户活动检测要求"]。
