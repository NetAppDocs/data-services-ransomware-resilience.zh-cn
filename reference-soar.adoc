= 为 NetApp Ransomware Resilience 集成 SOAR 攻略
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Ransomware Resilience 提供安全编排、自动化和响应 (SOAR) 攻略，使您能够自动执行威胁响应等任务。

Ransomware Resilience 攻略提供以下功能：

* 阻止用户*
* 创建卷的快照
* 利用威胁情报丰富 IP 地址
* 丰富给定 agent 和系统的存储信息
* 使卷脱机以进行事件响应
* 测试连接性
* 查看富化作业的状态


{asterisk} 只有 Splunk 和 Google SecOps SOAR 剧本才支持屏蔽用户。您必须在受支持的系统上配置 link:concept-user-behavior["用户行为活动检测"] 才能阻止用户。



== 操作手册
